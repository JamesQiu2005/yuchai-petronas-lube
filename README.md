# 玉柴马石油 · 特种润滑解决方案（演示站点）

面向销售演示的静态官网，托管于 GitHub Pages。

## 页面结构

| 页面 | 说明 |
| --- | --- |
| `index.html` | 首页：温度带动效、公司简介、核心优势、行业方案入口、立得系列、案例预览 |
| `solutions/steel.html` | 钢铁冶金：九大工序选型表 + 轧钢专项速查 + 通用润滑剂选型总表 |
| `solutions/precision.html` | 精密传动与机器人：立得 X3/X5 系列 + 15 类应用选型对照 |

| `solutions/automotive.html` | 汽车工业：十大部件系统润滑建议总表 |
| `solutions/tbm.html` | 盾构装备：全机润滑点位对照与国产替代 |

| `about.html` | 关于我们：企业概况、认证、服务支持、联系方式 |

## 上线前待办

1. **联系方式**：`about.html` 联系区与页脚中的电话/邮箱为占位文本，请替换为实际信息；
2. **客户案例**：`cases.html` 三个案例为示例数据（页面内已标注"示例"），取得客户授权后替换为真实数据；
3. **实景图片**：目前配图为原创矢量插画（无版权风险）。如需替换为工厂/产品实拍图，将图片放入 `assets/img/` 并替换对应 `<svg>`/`<img>` 即可——请勿使用带水印或未授权的图库图片。

## 修改内容

直接编辑对应 HTML 文件后提交推送即可，GitHub Pages 会在约 1 分钟内自动更新：

```bash
git add -A && git commit -m "更新内容" && git push
```

样式在 `assets/css/style.css`，交互（导航、滚动动效、温度带）在 `assets/js/main.js`，均无构建步骤、无外部依赖（不依赖任何 CDN，国内可正常访问）。
